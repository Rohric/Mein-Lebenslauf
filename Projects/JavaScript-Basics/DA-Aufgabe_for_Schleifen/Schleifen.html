Aufgabe 1 : Summe der Elemente eines Arrays

Implementiere eine Funktion namens sumArray(array), die mit einer for-Schleife die Summe der Elemente im Array berechnet und zurückgibt.

Aufgabe 2 : Ausgabe einer Zahlenreihe

Implementiere eine Funktion namens printNumbers(n), die mit einer for-Schleife die Zahlen von 1 bis n in aufsteigender Reihenfolge ausgibt.
Test:
printNumbers(5); // 1, 2, 3, 4, 5
printNumbers(3); // 1, 2, 3

Aufgabe 3 : Rückwärts laufende for-Schleife

Implementiere eine Funktion namens printNumbersReverse(n), die mit einer for-Schleife die Zahlen von n bis 1 in absteigender Reihenfolge ausgibt.
Test:
printNumbersReverse(5); // 5, 4, 3, 2, 1
printNumbersReverse(3); // 3, 2, 1

Aufgabe 4 : Nur jedes dritte Element ausgeben

Implementiere eine Funktion namens printEveryThirdElement(array), die mit einer for-Schleife jedes dritte Element des Arrays ausgibt (Index 0, 3, 6, …).
Test:
printEveryThirdElement([1, 2, 3, 4, 5, 6, 7, 8, 9]); // 1, 4, 7
printEveryThirdElement(['a', 'b', 'c', 'd', 'e', 'f']); // 'a', 'd'

Bonusaufgabe : Prüfung auf Primzahlen

Implementiere eine Funktion namens isPrime(n), die mit einer for-Schleife prüft, ob n eine Primzahl ist (nur durch 1 und sich selbst teilbar). Gib true zurück, wenn Primzahl, andernfalls false.
Test:
console.log(isPrime(7)); // true
console.log(isPrime(4)); // false
